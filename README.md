TMDbAPIPHP
=======

PHP library for retrieving movie or tv show from TMDb API.<br>

At this moment this is working:<br>

Search (for persons and movie/tv)<br>
Movie class (to retrieve data from a specific movie)<br>
Person class (to retrieve data from a specific person)<br>
Tv class (to retrieve data from a specific tv series incl seasons and episodes)


Credits to imdbphp, TMDbAPIPHP is loosly based on it.
