TMDbAPIPHP
=======

PHP library for retrieving movie or tv show from TMDb API.<br>

At this moment this is working:<br>

Search (for persons, movie, tv, keyword, company and collection)<br>
Movie class (to retrieve data from a specific movie incl watchproviders)<br>
Person class (to retrieve data from a specific person)<br>
Tv class (to retrieve data from a specific tv series incl seasons, episodes and watchproviders)


Credits to imdbphp, TMDbAPIPHP is loosly based on it.
